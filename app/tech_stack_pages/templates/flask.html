Flask

Flask is a helpfull Python library which allows us to create front-end applications.

Here I will provide some highlights on creating and testing Flask app.

Creating pages

You have multiple ways to create your pages. Flask allows us to configure paths of the app to perform specific actions based on what path you follow. You can display text content for the path. You can also display HTML content multiple ways: by writing HTML code inline or using HTML page templates for bigger projects.

Testing pages

After creating your website you can write tests to ensure your application runs properly. We can do it using Pytest. It allows to configure context variable and other parameters and then check on the state of the page. We can test for the correct page status, assert specific text or HTML value and various other methods.
