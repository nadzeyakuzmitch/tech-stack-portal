OOP theory and principles

Abstraction in a calculator projects hides unnecessary details, only showing relevant information for the user. The idea of encapsulation is to keep classes separated and prevent them from having tightly coupled with each other. Encapsulation keeps the data and the code safe from external interface. By using inheritance you can acquire some properties of another object. Convention denotes the new class as child class, and the one that it inherits from is called parent class or superclass. Polymorphism can be implemented by designing a generic interface, which provides generic methods for a certain class of action and there can be multiple classes, which provides the implementation of these generic methods.
